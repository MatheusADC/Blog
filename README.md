# <img src="https://github.com/user-attachments/assets/caabfdf0-0f9e-44a3-8200-c6579fe87887" alt="Description icon" width="28"> Description
The repository stores code intended for use in a blog that uses the **Next.js** framework.

# <sub><img src="https://img.icons8.com/?size=100&id=MWiBjkuHeMVq&format=png&color=000000" alt="Nextjs icon" width="36"></sub> Commands

### Create a project
```
npx create-next-app@latest
```

### Run a project
```
npm run dev
```

### Uninstall Tailwind CSS version 4
```
npm uninstall tailwindcss
```

 ### Install Tailwind CSS version 3 and its helpers
 ```
npm install tailwindcss@3 postcss autoprefixer
```

### Initialize Tailwind
```
npx tailwindcss init -p 
```

### Install shadcn
```
npx shadcn@latest init
```

### Add a Button Component
```
npx shadcn@latest add button
```
